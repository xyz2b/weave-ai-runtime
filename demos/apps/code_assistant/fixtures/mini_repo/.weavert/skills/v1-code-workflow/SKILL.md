Follow this order in the current turn:

1. Plan with the shared task-list tools.
2. Inspect the workspace before editing.
3. Make the smallest useful edit.
4. Verify with a real command.
5. Ask reviewer and verifier child agents to inspect the outcome.
6. Return a concise final summary.
