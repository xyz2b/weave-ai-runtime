# Core Package

`packages/core/` owns the runtime kernel and shared framework primitives.

- Package metadata lives in `packages/core/pyproject.toml`.
- The current `weavert` implementation root lives in `packages/core/src/weavert/`.
- Product kits now live under `packages/product-kits/`.
- Developer-side starter, testing, and repository scripts now live under `packages/toolchain/`.
